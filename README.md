# Bitcoin Market Sentiment vs Trader Performance Analysis

## Project Overview

This project investigates how Bitcoin market sentiment influences trader performance. Using the Fear & Greed Index and Hyperliquid historical trading data, the analysis explores profitability, trading activity, and trader behavior under different market conditions.

## Objective

* Analyze trader performance across Fear, Greed, and Neutral market conditions.
* Compare average profit and total profit by sentiment.
* Identify profitable trading patterns.
* Generate actionable insights from market sentiment data.

## Datasets

### Fear & Greed Index Dataset

* Date
* Classification
* Sentiment Value

### Hyperliquid Historical Trading Dataset

* Account
* Coin
* Execution Price
* Size USD
* Side
* Closed PnL
* Timestamp

## Methodology

1. Data Collection
2. Data Cleaning
3. Date Standardization
4. Dataset Merging
5. Exploratory Data Analysis (EDA)
6. Visualization
7. Insight Generation

## Key Analyses

* Trade Count by Sentiment
* Average Profit by Sentiment
* Total Profit by Sentiment
* Top Profitable Coins
* Buy vs Sell Behavior Analysis

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Conclusion

The results indicate that trader performance varies across different sentiment categories. Market sentiment appears to influence profitability and trading activity, suggesting that sentiment indicators can be useful tools for trading and risk management.
